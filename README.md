## 🎮 Description
VIDI is a psychological horror and puzzle game inspired by observation duty-style gameplay.  
The player monitors multiple rooms through security cameras and must detect and report anomalies before they escalate.

## 🧠 Gameplay
Players switch between camera feeds to observe different rooms.  
If any anomaly is detected, it must be reported immediately.  
Failing to detect anomalies in time may result in unexpected consequences.

## ✨ Features
- Camera-based room monitoring system
- Atmospheric glitch effects and immersive sound design
- Variety of anomalies with unique behaviors
- Different anomaly types:
  - Lethal anomalies
  - Jumpscare-triggering anomalies
- Increasing tension and psychological pressure

## 🛠️ Technologies Used
- Unity (Game Engine)
- C#

## 📌 Project Status
This project was developed as a learning experience to explore core game development concepts in Unity.

Implemented systems include:
- Camera switching system
- Anomaly detection logic
- Basic audio-visual atmosphere design
